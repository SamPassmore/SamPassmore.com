Kinbank: A database of Kinship terminology media watch

Fri, Jul 7, 2023 One-minute read

I recently published the main output from my PhD - Kinbank. Kinbank is a database of more than 1,200 kinship terminology from languages around the world. This post is a place to record the media reporting on the paper.

You can interact with the dataset here: www.kinbank.net

You can read the paper here: https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0283218
